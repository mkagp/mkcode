# Repository inventory

This inventory is based on manifests, imports, startup/build paths, tests, and
release workflows at commit `ecb35f75839925dd1ac6f854efeef5c9e291d11b`.
“Remove later” never means “safe to delete now”; the prerequisite column names
the consumer isolation required first.
